### Ansible роль для автоматического расширения LVM раздела

Этот плейбук проверяет наличие свободного пространства на диске. Если больше 100мб, расширит или создаст новый раздел, расширит группу томов, логический том  и файловую систему автоматически

шаблон переменных в файле `defaults/main.yml`:

```yaml
root_ext_dev: /dev/sda # имя устройства
root_ext_vg: rl_rocky9 # имя группы томов
root_ext_lv: root # имя логического тома
```

add_new_partition.yml - создает новый раздел

resize_ex_partition.yml - расширяет существующий раздел
